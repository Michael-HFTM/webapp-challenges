# Form Bugs — BugHunt

**Kurstag:** 10 — Signal Forms & Validierung
**Typ:** BugHunt (Warm-up)
**Dauer:** 15 Min.
**Bugs:** 2
**Format:** Einzelarbeit, PR mit CodeGuardian-Review

---

## Ausgangslage

Eine Angular-App mit einem Registrierungsformular hat Probleme. Das Formular verwendet `signalForm()` und Custom Validators — aber irgendetwas stimmt nicht. Die Validierung greift nicht korrekt und/oder die Formulardaten werden falsch verarbeitet.

## Aufgabe

Finde und fixe die **2 Bugs** im Formular:
- Die Bugs betreffen `signalForm()`, Form Fields und/oder Validators
- Das Formular soll nach dem Fix korrekt validieren und die Daten richtig verarbeiten

## Hinweise

- Teste das Formular im Browser — wann wird der Submit-Button aktiv?
- Prüfe die Validator-Logik — stimmen die Bedingungen?
- Achte auf die Signal-based Forms API

## Abgabe

Fork → Fix → Pull Request → CodeGuardian-Review anfordern (✨)
